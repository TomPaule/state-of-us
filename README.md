# The State of Us — Societal Health Index

A civic intelligence platform tracking twelve vital signs of societal health.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## What's built

- All 12 rings on the homepage with animated arc gauges
- Lives Lost ring — complete proof of concept with 7 categories, per-datapoint charts, broken incentives, actions, proximity to solutions, and recent developments
- Policy Action Tracker — 7 policy actions with per-ring breakdowns
- Navigation, trust banner, methodology statement

## Tech stack

Next.js 14 · TypeScript · Tailwind CSS · Recharts
